    .a11y-hidden {

이 부분이에요! 같이한번 해볼까요?

  • HTML 코드
</section>
<section  class="term">
<h2  class="term-heading">웹 관련 용어</h2>
<dl  class="term-list">
<dt  class="term-list-subject">웹 표준 이란?</dt>
<dd  class="term-list-thumbnail">
<img  src="./images/web_standards.gif"  alt="W3C"  />
</dd>
<dd  class="term-list-brief">
W3C 단체에서 규정한 웹 기술 사양에 대한 규칙을 말하며 표준 규격은...
</dd>
</dl>
</section>

dl, dt, dd,,,까먹었어요,,, 뭐였죠…?

  • <dl> 태그는 definition list(정의목록)의 약자로, 사전처럼 용어를 설명하는 목록을 만듭니다.

  • definition term(정의 용어)의 약자로, 정의되는 용어의 제목을 넣을 때 사용합니다.
  • definition description(정의 설명)의 약자로, 용어를 설명하는데 사용합니다. * 출처: https://aboooks.tistory.com/211 감사합니다.

  • CSS 코드

  • .term {
    margin-top: 20px; 가장 바깥 클래스인 term에 윗쪽으로 20마진
    border: 1px solid #aaa; 테두리 만들어주고
    border-radius: 5px; 5픽셀 겉테두리 깎아주고
    padding: 10px 15px; 좌우 상하로 10 15 픽셀씩 패딩을 입혀줍니다.
    background: linear-gradient(#ccc, #eee); 배경은 그라디언트 #ccc -> #eee로 줍니다.
    }
    .term-heading { 제목 클래스는
    font-size: 1.6rem; 1.6배의 글자크기를 가지구여
    font-weight: 700; 두꺼운 글씨체를 씁니다.
    }
    .term-list { 용어 리스트
    margin-top: 10px; 윗쪽 마진 10
    }
    .term-list::after { 가상 생성자 이거 중요합니다.
    content: “”; 가상으로 내용이 없는 생성자를 term-list클래스 생성자 뒤에 만듭니다
    display: block; 이 생성자의 display는 block이구요
    clear: both; 이 블록요소가 float되있는 요소를 감쌀 수 있도록 설정합니다. 요고 중요합니다.
    }
    !이 뒤의 float요소들이 마구 나오는데 걔네들은 float요소이기 때문에 특별한 처리를 해주지 않으면 박스 .term 위로 떠버리게 될 겁니다 예를 들면 이렇게되는거죠

그렇기 때문에 .term-list 의 뒤에 가상으로 블럭요소를 하나 만들고 그 블럭요소가 float를 받을 수 있도록 좌우로 clear 즉 clear: both를 주는 것입니다. 이 때 이 가상 블럭요소는 float요소가 아니기 때문에 이 요소자리까지 term 박스가 감쌀 수 있게 되는 것이죠 이제 list 요소들을 볼까요? 

.term-list-subject {
float: right;
width: 145px;
margin-top: -5px; 탑 마진이 +일 때는 컨텐츠가 아래로 내려가는 성질이 있습니다. 반대로 - 라면 컨텐츠가 올라가는 효과를 줍니다.
color: #296897;
}
.term-list-thumbnail {
float: left; 왼쪽정렬
}
.term-list-thumbnail  img { 이것은 썸네일 정렬인데...사실 왜있는지 모르겠습니다 없어도 그대로더라구요 없어도 될 것 같아요 제 생각엔 그래요.
vertical-align: top; 
}
.term-list-brief {
width: 145px; 좌우 길이를 주고
float: right; 오른쪽 정렬
line-height: 1.5; 한줄 높이
margin-top: 5px; 탑마진
}
