* css 코드
    * .header {
background-color: aquamarine; 색은 구분될 수 있게 아쿠아마린
position: relative; 포지션 relative로 설정
padding: 0  30px  50px; 위 좌우 아래의 패딩을 각각의 값으로 줍니다 
border-radius: 0  0  15px  15px; 이건 둥글게 깎는 작업입니다 좌상단 우상단 우하단 좌하단을 수치만큼 둥글게 깎습니다.
}
.logo {
position: absolute; 값을 조정할 수 있는데요 relative 한 부모컨테이너를 기준으로 움직입니다. 현재 부모컨테이너가 header고 relative값을 줬으니 header를 기준으로 움직일 것입니다
top: 47px; header를 기준으로 해당 수치만큼 이동
left: 65px;
}
.member {
// position: relative; 생각해보면 멤버 클래스는 relative를 가질 이유가 없습니다.
text-align: right; ul태그는 기본적으로 block성질입니다. 오른쪽으로 텍스트 정렬해줍니다.
right: -10px; 역시 부모인 header기준으로 -10만큼 이동해주고
background-color: azure; 구분하기위해 색을 줍니다
}
.member  li { 멤버 안에있는 li태그들을 총괄합니다
text-align: right; 오른쪽 정렬해주구요
font-size: 1.4rem; 기본적으로 설정한 폰트의 1.4배로 출력합니다
text-transform: uppercase; 대문자출력하구요
padding: 2px  0; 패팅은 위아래 2px입니다
background-color: bisque; 색깔 주구요
display: inline-block; 블록성질을 가진 인라인 텍스트입니다. 줄바꾸면 안되기 때문에 인라인 성질을 줬습니다.
}
.memeber  a {
display: inline-block; //솔직히 여기는 설정안해도 된다고 생각하는데요 아무문제없을것입니다 li태그 안서 뭐 움직일수도 없으니까요 그냥 일단두구요
padding: 8px  10px; 위아래 양 옆 패딩 이렇게 줍니다
}
