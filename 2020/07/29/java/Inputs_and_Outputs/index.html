Input and Output

자료의 입출력

출력 메소드

  • 포맷 문자열을 이용한 문자열/기본형 출력
  • 객체를 직접 출력

    PrintStream의 출력 메소드

    메소드 메소드 선언 설명
    print() public void print(Object obj) Obj를 스트림으로 출력
    println() public void println(Object x) x를 스트림으로 출력하고 줄바꿈
    printf() public PrintStream printf(String format, Object …args) Format에 맞추어 args를 출력한다.

지시자 자릿수 표현

표현 설명
%nd 최소 n칸을 사용하고 숫자를 오른쪽 정렬
%-nd 최소 n칸을 사용하고 숫자를 왼쪽 정렬
%0nd 최소 n칸을 사용하고 빈칸은 0으로 채움
%n.mf 최소 n칸을 사용하고 소수점 이하 m자리까지 표현

입력 메소드

*Scanner 클래스를 이용하여 입력받을 수 있다.

import java.util.Scanner;

...

Scanner scanner = new Scanner(System.in);
  • next(): 공백을 기준으로 한 단어씩 입력받는다
  • nextLine(): 한 줄 전체를 입력받는다.
  • nextInt(): int 값을 입력받는다.
  • nextDouble(): double 값을 입력받는다.
  • close(): 입력스트림을 종료한다.
