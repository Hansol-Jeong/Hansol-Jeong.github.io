DataType

자료형

  • 자료형은 기본형과(Primitive Type), 과 참조형 (Reference Type)이 있다.
  • 기본형 자료형에 는 정수형, 실수형, 문자형, 논리형이 있다
  • 참조형에는 대표적으로 문자열 데이터가 있다.

기본형

종류 자료형 크기(byte)
문자 char 2
정수 byte,short,int,long 1,2,4,8
실수 float,double 4,8
논리 boolean 1
  • 정수의 값 표현 범위
자료형 범위
byte -2^7 ~ 2^7-1
short -2^15 ~2^15-1
int -2^31~ 2^31-1
long -2^63~ 2^63-1

  • 실수형의 값 표현 범위

    자료형 범위 m e
    float -m * 10^e ~ m * 10^e 23 8
    short -m * 10^e ~ m * 10^e 52 11

    참조형

    값을 직접 저장하지 않고, 값이 저장된 주소를 저장 ex) Date birthDay = new Date();

    여러가지 자료형

    • 정수형
    • 실수형
    • 문자형
    • 논리형
    • 문자열

업캐스팅 다운캐스팅

업캐스팅

  • 자료의 범위가 더 큰 자료형으로 변환하는 경우

    • 묵시적, 명시적 방법 모두 사용 가능

      byte x = 10;
      int y = (int)x;
      int z = x;

다운캐스팅

  • 값의 범위가 더 작은 자료형으로 변환하는 경우

    • 명시적인 방법으로만 캐스팅 가능

      int x = 1024;
      byte z = (byte)x;
