Strings

문자열

문자열의 특징

  • 클래스로 구현되어 있다.

  • 내부에 문자의 배열로 데이터가 구현되어 있다.

  • 한번 만든 문자열은 변하지 않는다. (Immutable) (Strings)

    • StringBuilder(동기화 x 빠름), StringBuffer(동기화 o, 느림)사용

      문자열의 생성

  • new 키워드를 이용하여 생성

    • String string = new String("문자열 생성");

  • new 키워드 없이 생성 (권장)

    • String string = "문자열 생성";

      문자열 메소드

  • length(): 문자여르이 길이를 출력
  • charAt(int index): index번 쨰 위치한 문자 출력
  • indexOf(char ch): ch가 위치한 iㅜdex출력, 없을 시 -1
  • equals(Object ob): ob와 비교한 결과 출력
  • equalsIgnoreCase(String string): 대소문자 구분없이 string과 비교 결과 출력
  • replace(char oldChar, char newChar): oldChar를 찾아 newChar로 변경된 문자열 출력
  • substring(int beginIndex, int endIndex): 문자열을 beginIndex부터 endIndex -1까지 잘라서 출력
  • trim(): 문자열 좌우 공백을 없앤 결과를 출력
  • matches(String regex): 문자열을 정규표현식 regex확인 결과 출력
  • split(String regex): 문자열을 정규표현식 형태로 나눈 후 배열로 출력

도전과제

  • 정규표현식
  • StringBuilder
  • StringBuffer

N-D_Arrays

N-D Arrays

다차원 배열의 정의

  • 배열을 담고 있는 배열을 다차원 배열이라고 한다.
  • 수학의 선 -> 면 -> 공간 등과 동일한 개념

다차원 배열의 생성

  • 다차원 배열의 선언

    • int[] array2D[];
    • int[][] array2D;
    • int array2D[][];

다차원 배열의 생성

  • 여러 차원의 배열을 동시에 생성

     int[][] array2D = new int[5][10];

  • 상위 차원의 배열부터 생성

     int[][] array2D = new int[5][];
 for (int i = 0; i < array2D.length; i++) {
 array2D[i] = new int[10];

    }

  • 다차원 배열의 초기화

       int[][] array2D = {{3, 5, 1, 20, 65},
                   {5, 2, 6}, // length may vary
                   {10, 3, 5, 67, 3}};

배열과 반복문

  • 인덱스를 이용한 배열 접근
   int[][] array2D = new int[10][10];
    for (int i = 0; i < array2D.length; i++) {
      for (int j = 0; j < array2D[i].length; j++) {
        array2D[i][j] = i * j;
      }
    }

java►til►Arrays

Arrays

배열 (Arrays)

배열의 특성

  • 하나의 변수로 여러개의 값을 다룰 수 있다.
  • 동일한 자료형의 값만 다룰 수 있다.
  • 한번 생성된 배열의 크기는 변하지 않는다.
  • 배열에 속한 값은 메모리에 연속적으로 위치한다.

배열의 생성

  • 배열의 선언

    • int[] intArray;// 자료형[] 변수명; recommended
      int integerArray[];// 자료형 변수명[]; old c-style

    배열의 생성/초기화

  • 생성 후 값 할당

    • int[] intArray = new int[10];
      intArray[0] = 4;
      intArray[1] = 10;
      • int[] intArray;
        intArray = new int[]{4, 5, 1, 2, 5};
    • 생성과 동시에 값 할당
    • int[] intArray = {3, 5, 1, 20, 65};
    • int[] intArray2 = new int[]{4, 6, 2, 3, 4};

배열과 반복문

  • 인덱스를 이용한 배열 접근

    • float[] floatArray = new float[10];
      for (int i = 0; i < floatArray.length; i++) {
      floatArray[i] = i * 0.5;
      }

  • 향상된 for 문을 이용한 배열 접근

    • int[] intArray = {4, 5, 1, 2, 7, 5};
      for (int value: intArray) {
      System.out.println(value);
      }

배열의 크기 변경

  • 배열의 크기는 변경할 수 없으므로 새로운 배열을 만들고 데이터를 옮겨야 한다.

    • int[] src = {1, 2, 3, 4, 5};
      int[] dst = new int[10];
      for(int i = 0; i < source.length; i++) {
      dst[i] = src[i];
      }
    • int[] src = {1, 2, 3, 4, 5};
      int[] dst = new int[10];
      System.arraycopy(src, 0, dst, 0, src.length);

Loops

반복문

반복문의 일반적인 구성

  • 초기화

  • 조건식

  • 실행문

  • 증감식

    다양한 반복문

    for 문

    for(초기화; 조건식; 증감식) {

    // 실행문

    }

while 문

  • while 문의 경우 실행문이 한번도 실행되지 않을 수도 있다.

    while(조건식) {
    // 실행문
    // 증감식
    }

    do ~ while 문

  • do ~ while 문의 경우 실행문은 무조건 한번 이상 실행된다.

    do {
    //실행문
    //증감식
    } while(조건식)

반복문 제어

break 문

  • 반복문을 곧바로 종료한다.

    while (조건식) {
    if (종료조건) {

    break;

    }
    //실행문
    //증감식
    }

continue 문

  • 반복문을 곧바로 다음 반복으로 건너 뛴다.
  • while 문의 경우 증감식이 실행되지 않을 수 있다.

*

while (조건식) {
  if (제어조건) {
    //증감식
    continue;
  }
  //실행문
  //증감식
}

label

  • 중첩 반복문에서 어떤 반복문을 제어할지 결정

    • loop1: for (int i = 0; i < 10; i++) {
      loop2: for (int j = 0; j < 10; j++) {
      if (j == 3) {
      break;
      // break loop1;
      // break loop2;
      // continue;
      // continue loop1;
      // continue loop2;
      }
      }
      }

Conditional_Statements

조건문

조건문의 구성

  • 조건식, 실행문

    다양한 조건문

if 계열 조건문

if 문

if(조건식)
  // 실행문

if(조건식) {
  // 실행문
}

if~else문

if(조건식) {
  // 조건식이 true일때 실행할 실행문
} else {
  // 조건식이 false일때 실행할 실행문
}

if ~ else if ~ else 문

if(조건식1) {
  // 조건식1이 true일때 실행할 실행문
} else if(조건식2) {
  // 조건식1이 false이고, 조건식2가 true일때 실행할 실행문
} else {
  // 모든 조건식이 false일 때 실행할 실행문
}

Nested if 문

if(조건식1) {
  if(조건식2) {
    // 조건식1과 조건식2가 true일때 실행할 실행문
  } else {
    if(조건식3) {
      // 조건식1과 조건식3이 true이고 조건식2가 false일때 실행할 실행문
    }
  }
}

switch ~ case

  • 조건식과 비교할 대상이 여럿일 때 사용

    • 조건식의 출력은 정수값, 문자열, 열거형 등 정확한 값을 리턴
  •  switch(조건식) {
 case 비교값1:
   // 실행문1
   break;
 case 비교값2:
   // 실행문2
   break;
 default:
   // 실행문3

    }

  • Fall-through

    • break문을 사용하지 않고 다음 case문을 실행시키는 방법

  • switch(조건식) {

    case 비교값1:
  // 실행문1
case 비교값2:
  // 실행문2
default:
  // 실행문3

    }
    // 비교값1에 해당할 경우: 실행문1, 실행문2, 실행문3 실행
    // 비교값2에 해당할 경우: 실행문2, 실행문3 실행
    // 해당하는 값이 없을 경우: 실행문3 실행

Input and Output

자료의 입출력

출력 메소드

  • 포맷 문자열을 이용한 문자열/기본형 출력
  • 객체를 직접 출력

    PrintStream의 출력 메소드

    메소드 메소드 선언 설명
    print() public void print(Object obj) Obj를 스트림으로 출력
    println() public void println(Object x) x를 스트림으로 출력하고 줄바꿈
    printf() public PrintStream printf(String format, Object …args) Format에 맞추어 args를 출력한다.

지시자 자릿수 표현

표현 설명
%nd 최소 n칸을 사용하고 숫자를 오른쪽 정렬
%-nd 최소 n칸을 사용하고 숫자를 왼쪽 정렬
%0nd 최소 n칸을 사용하고 빈칸은 0으로 채움
%n.mf 최소 n칸을 사용하고 소수점 이하 m자리까지 표현

입력 메소드

*Scanner 클래스를 이용하여 입력받을 수 있다.

import java.util.Scanner;

...

Scanner scanner = new Scanner(System.in);
  • next(): 공백을 기준으로 한 단어씩 입력받는다
  • nextLine(): 한 줄 전체를 입력받는다.
  • nextInt(): int 값을 입력받는다.
  • nextDouble(): double 값을 입력받는다.
  • close(): 입력스트림을 종료한다.

DataType

자료형

  • 자료형은 기본형과(Primitive Type), 과 참조형 (Reference Type)이 있다.
  • 기본형 자료형에 는 정수형, 실수형, 문자형, 논리형이 있다
  • 참조형에는 대표적으로 문자열 데이터가 있다.

기본형

종류 자료형 크기(byte)
문자 char 2
정수 byte,short,int,long 1,2,4,8
실수 float,double 4,8
논리 boolean 1
  • 정수의 값 표현 범위
자료형 범위
byte -2^7 ~ 2^7-1
short -2^15 ~2^15-1
int -2^31~ 2^31-1
long -2^63~ 2^63-1

  • 실수형의 값 표현 범위

    자료형 범위 m e
    float -m * 10^e ~ m * 10^e 23 8
    short -m * 10^e ~ m * 10^e 52 11

    참조형

    값을 직접 저장하지 않고, 값이 저장된 주소를 저장 ex) Date birthDay = new Date();

    여러가지 자료형

    • 정수형
    • 실수형
    • 문자형
    • 논리형
    • 문자열

업캐스팅 다운캐스팅

업캐스팅

  • 자료의 범위가 더 큰 자료형으로 변환하는 경우

    • 묵시적, 명시적 방법 모두 사용 가능

      byte x = 10;
      int y = (int)x;
      int z = x;

다운캐스팅

  • 값의 범위가 더 작은 자료형으로 변환하는 경우

    • 명시적인 방법으로만 캐스팅 가능

      int x = 1024;
      byte z = (byte)x;

Variable

변수

  • 변수는 변하는 수

  • 수학의 변수와 유사하다

  • 변수란 것은 변수타입이 존재하고, 변수에는 변수명, 값을 할당할 수 있다.

    사용 예

  • int y,z, value; 여러개도 동시 선언 가능

  • int valueOne =10, valueTwo =20; 이렇게 한줄에도 가능

  • int int; 이 경우는 되지 않는다. 이미 예약어로 잡혀있어서, 자바에서 문법적으로 쓰는 키워드는 변수명으로 사용이 불가하다.

  • int intOne; 이렇게는 가능하다.

  • int 심지어_한글도_됩니다_하지만_쓰지_않습니다;

  • int 4four; 숫자로 시작할 순 없습니다.

  • int _4four; 숫자로 쓰고싶다면 _로 시작하면 됩니다. _는 특수문자가 아닙니다.

  • int fe$the; 특수문자는 되지 않지만 $는 됩니다. 하지만 쓰지 않습니다.

    CodeConvention

    변수명을 어떠한 식으로 쓰면 좋은가

    camelCase

    첫번째 소문자 그다음 띄어쓰기 대신에 대문자

    • int valueOfComputer;

      PascalCase

      UpperCamelCase라고도 한다
    • PascalCase는 변수이름으론 쓰지 않습니다.
